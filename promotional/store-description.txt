=== 详细描述 ===

📋 Copy as Markdown - 智能网页内容转换工具

复制网页内容时自动转换为 Markdown 格式，特别优化了数学公式的处理，是学术研究、技术文档整理的必备工具。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

✨ 核心功能

• 自动转换：复制即转换，无需额外操作
• 数学公式：完整支持 KaTeX、MathJax、MathML 格式
• 表格处理：HTML 表格自动转换为 Markdown 表格
• 代码块：保留语法高亮语言标识
• 链接转换：自动转换为 [文字](URL) 格式
• 图片支持：转换为 ![alt](src) 语法
• 删除线：支持 ~~删除线~~ 语法

━━━━━━━━━━━━━━━━━━━━━━━━━━━━

⌨️ 使用方法

1. 在网页上选择要复制的内容
2. 按 Ctrl+C 复制
3. 粘贴到目标位置：
   • Ctrl+V - 粘贴为富文本（适用于 Word、邮件）
   • Ctrl+Shift+V - 粘贴为 Markdown 纯文本（适用于 Typora、VS Code）
